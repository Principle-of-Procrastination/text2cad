01 · Baseline

环境基线与记录要求

每次验收必须记录精确版本，不用“最新版”代替版本号。

机器与工具

  • Windows 11 edition、version、OS build
  • SOLIDWORKS 2026 完整版本和 SP
  • Visual Studio Community 2026 完整版本
  • .NET Framework 4.8 SDK/Targeting Pack
  • .NET 10 SDK 与 Host runtime

构建与协议

  • commit、Add-in build、Host build
  • protocolVersion、schemaVersion
  • capability set 与 plan hash 算法
  • Interop DLL 来源路径和文件版本
  • SOLIDWORKS 模板、语言、单位和 configuration

硬性要求

  • Add-in、SOLIDWORKS 和 Interop 必须同为 x64。
  • 实现只使用 SOLIDWORKS 2026 API 表面和同版本 PIA；PIA 的来源路径和文件版本必须可追溯，不混用来历不明的 DLL。
  • SOLIDWORKS COM 调用只能经过 Add-in 的 CadDispatcher。
  • 所有几何测试仅在专用测试零件中执行，不使用生产模型。
  • 旧 src/SwCopilot 和 deploy 不属于当前验收对象。

03 · M1

确定性 CAD 闭环

Golden fixture

  • 中心矩形板：100 × 60 × 8 mm
  • 4 个直径 6 mm 的通孔
  • 孔中心距对应边各 10 mm
  • 单一实体
  • 草图、约束、驱动尺寸、拉伸和切除均为原生特征

人工检查

显示名称可使用 T2C_BasePlate、T2C_ThroughHoles 帮助检查，但名称不能作为唯一对象身份。稳定引用应包含文档身份、对象类型与 Persistent Reference ID 等证据。

04 · Negative & recovery

拒绝与恢复门禁

ID注入条件必须结果
NEG-01未知 operationValidator 拒绝，未触达 COM。
NEG-02零值、负值、NaN/Infinity、非法单位或越界尺寸拒绝且模型不变。
NEG-03确认后篡改计划planHash 不匹配，拒绝执行。
NEG-04Preview 后修改文档或切换活动文档revision/documentId 不匹配，确认失效。
NEG-05Host 断线SOLIDWORKS 稳定，不执行缓存计划。
REC-01测试构建在 operation 1 后注入失败返回结构化错误并恢复执行前状态。
REC-02Rebuild 或后置条件失败success=false，不得报告部分成功。
REC-03Undo/补偿无法证明恢复标记“需要人工检查”，停止后续操作。
!
Undo Object 不是数据库事务

只有 SOLIDWORKS 本身支持 Undo 的操作才可能被撤销。恢复后必须再次 rebuild，并比较执行前后状态；不能仅因调用 Undo 没有抛异常就报告恢复成功。

05 · Acceptance

退出标准

M0 通过

  • Add-in 可反复加载、卸载和重启。
  • Host 崩溃或断线不拖垮 SOLIDWORKS。
  • Named Pipe 完成版本协商和恢复。
  • 快照准确更新且不修改模型。
  • COM 调用只经过 CadDispatcher。
  • 全链路有 requestId 和结构化错误。

M1 通过

  • Preview 与取消绝不修改模型。
  • 用户确认不可变 planHash。
  • Stale/非法计划在 COM 前被拒绝。
  • 固定计划重复产生等价原生树。
  • 保存重开后仍可编辑并 rebuild。
  • 失败能证明恢复或要求人工检查。
06 · Troubleshooting

失败排查

Add-in 不出现在列表

检查新实现的 CLSID、注册工具、x64 配置和已记录的 SOLIDWORKS 2026 Interop 来源。不要使用旧 deploy/install.bat。

BadImageFormatException

检查 SOLIDWORKS、Add-in、依赖和注册工具是否混用了 x86/x64。

Interop 或 COM 类型加载失败

确认构建使用 SOLIDWORKS 2026 PIA，并核对当前安装目录下 api\redist 的来源、加载路径和文件版本。

Task Pane 不显示或重复

检查 ConnectToSW、CreateTaskpaneView3、图标路径、窗口生命周期和重复实例清理。

Host 无法连接

检查 Named Pipe ACL、会话令牌、协议版本、超时与 Host 日志；不得因此阻塞 SOLIDWORKS UI。

Snapshot 不更新

检查文档/选择事件订阅和 revision 生成逻辑。快照处理只能使用短生命周期 COM 引用。

尺寸或对象定位失败

稳定引用可能已失效。禁止退化为只按显示名称搜索；返回引用失效并要求重新规划。

本地化基准面失败

禁止硬编码 Front Plane 或“前视基准面”；使用对象类型、语义、模板契约或稳定引用。

Rebuild 失败

保留 operation 证据，执行恢复并返回 rollbackStatus；不得把部分几何创建报告为成功。

FeatureManager 只有 Imported1

M1 失败。验证目标是原生 sketch、dimension、extrude 和 cut，而不是导入体。

Undo 不完整

确认该 API 操作是否受 SOLIDWORKS Undo 支持；补充 compensation、执行前副本或更窄的 capability。
