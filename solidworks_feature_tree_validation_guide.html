验证目标

这一步只验证 SolidWorks API 能否写入原生建模特征，不验证自然语言或 AI 质量。

要生成的零件

一个 100 x 60 x 8 mm 底板，带 4 个直径 6 mm 的通孔。

要看到的结果

FeatureManager 里出现 AI_BasePlate_100x60x8 和 AI_4x_M6_ThroughHoles。

关键判断

右键特征应能 Edit Feature，右键草图应能 Edit Sketch。

不是这次要验证的东西

不要从 STEP 导入；如果树里只有 Imported1，说明没有验证到原生特征树。

操作步骤

全程在 SolidWorks 桌面版里完成，不需要 Visual Studio。

打开 SolidWorks

确认能正常进入主界面，并且当前 license 可用。

新建一个 Macro

在菜单里选择：

Tools > Macro > New

保存为 text2cad_feature_tree_test.swp。

粘贴宏代码

VBA 编辑器打开后，清空默认内容，把下面的完整代码复制进去。

运行 Macro

点击 VBA 编辑器里的运行按钮，或回到 SolidWorks 选择：

Tools > Macro > Run

检查 FeatureManager

运行成功后，左侧特征树里应该有原生拉伸和切除特征。请截图并按“回传信息”模板反馈。

VBA Macro 代码

点击右上角复制按钮。粘贴到 SolidWorks VBA Macro 的主模块里运行。

这个版本已按实机验证结果更新：优先选择中文 前视基准面，失败后遍历特征树选择第一个基准面；孔切除方向也已修正。
text2cad_feature_tree_test.swp 
Option Explicit

Dim swApp As SldWorks.SldWorks
Dim swModel As SldWorks.ModelDoc2
Dim swFeat As SldWorks.Feature
Dim ok As Boolean

Const MM As Double = 0.001

Sub main()

    Set swApp = Application.SldWorks

    Dim templatePath As String
    templatePath = swApp.GetUserPreferenceStringValue(swUserPreferenceStringValue_e.swDefaultTemplatePart)

    If templatePath = "" Then
        MsgBox "No default part template found. Please set a default Part template in SOLIDWORKS."
        Exit Sub
    End If

    Set swModel = swApp.NewDocument(templatePath, 0, 0, 0)

    If swModel Is Nothing Then
        MsgBox "Failed to create new part document."
        Exit Sub
    End If

    swModel.ClearSelection2 True

    ' 1) Create base sketch on Front Plane
    ok = SelectFrontPlane()
    If Not ok Then
        MsgBox "Failed to select Front Plane. If your SOLIDWORKS UI is localized, update the plane name in SelectFrontPlane()."
        Exit Sub
    End If

    swModel.SketchManager.InsertSketch True
    swModel.SketchManager.CreateCenterRectangle 0, 0, 0, 50 * MM, 30 * MM, 0

    Set swFeat = swModel.FeatureManager.FeatureExtrusion3( _
        True, False, False, _
        swEndCondBlind, 0, _
        8 * MM, 0, _
        False, False, False, False, _
        0, 0, _
        False, False, False, False, _
        True, True, True, _
        swStartSketchPlane, 0, False)

    If swFeat Is Nothing Then
        MsgBox "Failed to create base extrude."
        Exit Sub
    End If

    swFeat.Name = "AI_BasePlate_100x60x8"

    swModel.ClearSelection2 True

    ' 2) Create four hole circles on the same plane
    ok = SelectFrontPlane()
    If Not ok Then
        MsgBox "Failed to select Front Plane for holes."
        Exit Sub
    End If

    swModel.SketchManager.InsertSketch True

    swModel.SketchManager.CreateCircleByRadius 40 * MM, 20 * MM, 0, 3 * MM
    swModel.SketchManager.CreateCircleByRadius -40 * MM, 20 * MM, 0, 3 * MM
    swModel.SketchManager.CreateCircleByRadius 40 * MM, -20 * MM, 0, 3 * MM
    swModel.SketchManager.CreateCircleByRadius -40 * MM, -20 * MM, 0, 3 * MM

    Set swFeat = swModel.FeatureManager.FeatureCut4( _
        False, False, False, _
        swEndCondThroughAll, 0, _
        10 * MM, 10 * MM, _
        False, False, False, False, _
        0, 0, _
        False, False, False, False, _
        False, True, True, True, True, False, _
        swStartSketchPlane, 0, False, False)

    If swFeat Is Nothing Then
        MsgBox "Failed to create hole cut."
        Exit Sub
    End If

    swFeat.Name = "AI_4x_M6_ThroughHoles"

    swModel.EditRebuild3
    swModel.ViewZoomtofit2

    DumpFeatureTree

    MsgBox "Done. Check the FeatureManager tree: native Boss-Extrude and Cut-Extrude features should exist."

End Sub

Function SelectFrontPlane() As Boolean
    ' Ensure the document is active and selection state is clean.
    swApp.ActivateDoc swModel.GetTitle
    swModel.EditRebuild3
    swModel.ClearSelection2 True

    ' Method 1: select the localized default front plane by name.
    If swModel.Extension.SelectByID2("前视基准面", "PLANE", 0, 0, 0, False, 0, Nothing, 0) Then
        SelectFrontPlane = True
        Exit Function
    End If

    ' Method 2: fall back to the first plane in the feature tree.
    Dim swFeat As SldWorks.Feature
    Set swFeat = swModel.FirstFeature
    Do While Not swFeat Is Nothing
        If swFeat.GetTypeName2 = "Plane" Then
            If swFeat.Select2(False, 0) Then
                SelectFrontPlane = True
                Exit Function
            End If
        End If
        Set swFeat = swFeat.GetNextFeature
    Loop

    SelectFrontPlane = False
End Function

Sub DumpFeatureTree()
    Dim f As SldWorks.Feature
    Set f = swModel.FirstFeature

    Debug.Print "---- FeatureManager Tree ----"

    Do While Not f Is Nothing
        Debug.Print f.Name & " | " & f.GetTypeName2
        Set f = f.GetNextFeature
    Loop
End Sub

成功标准

只要下面几项成立，就说明 SolidWorks API 路线可以保留原生特征树。

FeatureManager 出现命名特征能看到 AI_BasePlate_100x60x8 和 AI_4x_M6_ThroughHoles。
不是 Imported Body树里不应该只有 Imported1、Imported2 这种导入体。
特征可以编辑右键 AI_BasePlate_100x60x8 应该能看到 Edit Feature。
草图可以编辑右键孔对应的 sketch，进入 Edit Sketch 后能看到 4 个圆。
文件可以保存和重开保存为 .SLDPRT 后重新打开，特征树仍然存在。
如果这些都通过，后续产品可以走“自然语言 → 结构化 feature plan → SolidWorks API 执行 → .SLDPRT 原生特征树”的路线。

失败排查

优先看报错信息和 FeatureManager 结果，不要先改建模逻辑。

提示找不到 Front Plane

当前宏会先选 "前视基准面"，再遍历特征树选择第一个 Plane。如果仍失败，请把 SelectFrontPlane() 里的名称改成左侧树显示的实际基准面名称，或反馈完整特征树截图。

编译报 User-defined type not defined

在 VBA 编辑器里打开 Tools > References，勾选当前版本的 SOLIDWORKS type library，然后重新运行。

没有默认 Part Template

报 No default part template found 时，在 SolidWorks 设置里配置默认 Part template，或先手动新建一个 Part 再运行改造版宏。

FeatureManager 里只有 Imported1

这说明走了 STEP 导入或其他导入流程，不是这次 API 建模验证。请重新按本页 Macro 路线运行。

孔没有切穿或方向不对

这次实测把 FeatureCut4 的第一个参数从 True 改为 False 后可行。产品化时切除方向需要由执行器根据草图平面、实体方向和预期布尔结果自动判断。
